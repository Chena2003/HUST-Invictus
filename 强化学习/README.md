## 签到

平时基本不签到，但如果上课人数过少，可能会签到。

平时上课老师可能会布置课堂作业，让同学们现场写现场交（也算是一种签到😅）。

## 考试

强化学习不设置考试，考核形式是让同学组队做一个强化学习项目，进行课堂展示并写一个报告。
